# 🔐 Password Generator & Strength Checker (C Project)

## 📌 Description
This project is developed using the C programming language.  
It can generate random passwords and check password strength using different security conditions.

---

## 🎯 Features
- Generate random passwords
- Check password strength
- Colored console interface
- Menu-driven program
- Box-style UI

---

## 🛠️ Technologies Used
- C Programming Language
- GCC Compiler
- VS Code / CodeBlocks

---

## ▶️ How to Run

### Compile
gcc password_manager.c -o password

### Run
./password

(For Windows)
password.exe
